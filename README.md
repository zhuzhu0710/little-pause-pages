# Little Pause Pages - Website

## Project Overview

Little Pause Pages is a premium brand website for toddler-friendly coloring books and quiet-time printables designed for ages 1–4. The website features a warm, calm, and trustworthy design that resonates with parents and caregivers.

## Design System

### Brand Colors
- **Primary Cream**: `#F5F1E8` - Warm, calm background
- **Deep Navy**: `#1B2B4A` - Primary text color
- **Teal Accent**: `#4A9B8E` - Secondary accent, call-to-action elements
- **Coral Accent**: `#E8876A` - Primary accent, buttons, highlights
- **Soft Gold**: `#D4A574` - Tertiary accent, decorative elements

### Typography
- **Font Family**: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
- **Headings**: Bold (600 weight), Deep Navy color
- **Body Text**: Regular weight, 1.6 line-height for readability
- **Responsive**: Font sizes scale appropriately for mobile, tablet, and desktop

## How to Use

### Local Preview
1. Open any HTML file in a web browser
2. Navigate between pages using the header navigation
3. Test responsive design by resizing the browser window

### Customization
- **Colors**: Update color values in `styles.css` (search for hex codes)
- **Content**: Edit text directly in HTML files
- **Images**: Replace image files in the `assets/` folder
- **Fonts**: Modify font-family in `styles.css`

### Adding New Pages
1. Create a new HTML file following the existing template
2. Include the header, footer, and navigation
3. Add your content in the main section
4. Link to the new page from the navigation menu
